== Description ==

Simple way to add contact form on your wordpress site.Just add the shortcode as[just_contact_form] on the page to create contact form.It is lightweight and would not slow down your wordpress website.
Entered user details are sent to the registered email id.Entered user details are sent to the registered email id.


To add the contact form :
1.Add a page in which you want to display contact form for the user
2.Add the shortcode [just_contact_form] in the above page.The receiver email id ,success and failure messages can be updated in the setttings.

== Installation ==

1. Download plugin from "Plugins"=>Add new plugin.
2. Go to admin panel open "Plugins" and activate Just Contact.
3. Navigate to "settings"=>"Just Contact".Verify the details.

== Frequently Asked Questions ==
1.Go to Settings-->Just Contact
2.Receiver email address,Success message and Failure message can be edited on this page

"Receiver" is the email address of the person to whom the contact form messages will be emailed.By default this is the configured wordpress email address.
"Success message" is displayed after an email has been sent when user clicks the submit button.
"Failure message" is displayed if there is any problem in sending the email to the receiver's email address.
